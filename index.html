Making links

All types of links use an anchor or a

Types of Links

  • href: can use absolute or relative links
  • hreflang: specifies language of resource linked resource
  • rel: specifies relationship between current and linked document
  • target: specifies where to open, ex. _blank, _self, _parent, _top
  • title: specifies extra information about a link
  • download: specifies that the target will be downloaded

Ways to Link

You can use absolute URLs always, and can also use rel to show what kind of link it is, like this: 

example site

You can jump between sections of a page by using id tags like this: 

Table of Contents

          Click to jump to the First Topic
          Click to jump to the Second Topic

You can do this across the same website like this:

Remember, you can always look back in the First Topic for
              supporting information.

To link to a page on the same site, you can simply use the backslash / to link it: 

Text Here
            Text Here

Making a link with tel: dials a number like Skype or Facetime: 

Call us
